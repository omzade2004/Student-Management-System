# Student Management System

A command-line application built with Python and SQLite to manage student records efficiently.

## Features
- Add, view, update, and delete student records (full CRUD)
- Grade calculation with automatic result generation
- Modular code structure with clean separation of concerns
- Manages 100+ student records with SQLite database

## Tech Stack
- Python
- SQLite
- Modular Programming / OOP

## How to Run
1. Clone the repo
   git clone https://github.com/omzade2004/student-management-system
2. Navigate to the folder
   cd student-management-system
3. Run the app
   python main.py
